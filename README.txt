INSTRUKCJA: Jak wgrać stronę na hosting

1. Utwórz folder z plikami: index.html, style.css, script.js.
2. Spakuj folder do ZIP (jeśli hosting tego wymaga).
3. Zaloguj się na swój hosting (np. dhosting, home.pl, nazwa.pl).
4. Wejdź do menedżera plików / FTP i wgraj pliki do folderu publicznego (zwykle public_html).
5. Wejdź na swoją domenę – strona powinna się wyświetlić!
6. Aby edytować treści – otwórz plik index.html np. w Notepad++ lub VS Code i zmieniaj teksty.
